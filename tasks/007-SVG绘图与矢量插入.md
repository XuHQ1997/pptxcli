# 任务 007：SVG 绘图与矢量插入

## 目标

支持由 Agent 生成 SVG 内容，并将其作为矢量图插入到 PPT 的模板字段位置或非模板的任意位置，同时为兼容性问题提供可控的回退方案。

## 主要工作

- 调研 PowerPoint 对 SVG 插入、编辑和展示的兼容性边界
- 设计 SVG 输入方式，支持内联 SVG 字符串与外部 `.svg` 文件
- 支持将 SVG 作为图片类字段填充到模板中的指定响应位置
- 支持在非模板模式下按坐标、尺寸插入 SVG 到任意 slide 位置
- 设计插入策略，区分原生 SVG 插入、预渲染 PNG 回退、可选 EMF/WMF 回退
- 处理 viewBox、尺寸映射、透明背景、裁剪和对齐规则
- 建立预览与验收链路，确认生成结果在目标环境中的可用性

## 交付物

- SVG 插入能力说明
- `template fill` 对 SVG 字段的支持，或独立 `svg insert` / `draw svg` 命令
- SVG 定位参数与字段映射约定
- 兼容性说明与回退策略文档

## 验收标准

- Agent 可提供 SVG 内容并成功插入到模板字段位置
- Agent 可指定 slide 与几何位置，将 SVG 插入任意位置
- 在主目标环境中优先保留矢量效果
- 当目标环境对原生 SVG 支持不稳定时，可切换到明确的回退方案
