# 任务 002：PPT 结构解析与中间模型

## 目标

解析 PPTX 中的 slide、shape、text、image 等基础结构，建立统一中间模型。

## 主要工作

- 解包 pptx 并读取基础 OOXML 文件
- 建立 slide / shape / text / image 的数据模型
- 为每个对象生成稳定标识
- 输出便于调试的结构化 JSON

## 交付物

- PPT 解析模块
- 中间模型定义
- 调试输出命令或示例 JSON

## 验收标准

- 能读取一个 PPT 的页数与每页对象列表
- 能区分文本框、图片等主要对象类型
- 能输出稳定的对象 ID 与基础几何信息
