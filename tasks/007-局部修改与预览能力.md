# 任务 007：局部修改与预览能力

## 目标

支持替换指定位置的幻灯片，并导出指定页预览图，形成 Agent 闭环校验能力。

## 主要工作

- 设计 slide patch / replace 交互格式
- 支持在指定位置插入或替换模板实例
- 集成 PPT 转图片的预览链路
- 支持输出单页或多页预览结果
- 基于当前会话直接预览内存中的最新编辑结果或已保存结果

## 交付物

- `template modify` 或 `template patch` 命令
- `preview` 命令
- 预览输出说明

## 验收标准

- 能替换指定页并生成更新后的 PPT
- 能导出指定页预览图
- 预览结果可用于 Agent 二次检查
