# 任务 004：会话化服务与状态管理

## 目标

将 `pptx_cli` 扩展为单会话的 CS 模式工具，通过后台 server 复用 PPT 的加载、解析与编辑现场，支持多轮模板提取与内容创作。

## 主要工作

- 设计 `pptxcli init` / `pptxcli finish` 命令
- 启动单实例后台 server，并限制同一时间仅服务一个 client
- 在 server 内保存当前 PPT、解析结果、中间模型和编辑上下文
- 设计 server 地址发现机制，将监听地址写入 `pptxcli` 所在目录的状态文件
- 处理 server 存活检查、重复启动、异常退出和状态清理
- 定义前台 CLI 与后台 server 的通信方式

## 交付物

- `init` / `finish` 命令
- 单会话 server 原型
- 本地状态文件格式约定
- 基础会话生命周期文档

## 验收标准

- 执行 `pptxcli init --origin_file xxx.pptx` 后可成功启动 server 并加载 PPT
- 后续命令可在不重复传入完整上下文参数的情况下复用当前会话
- 执行 `pptxcli finish` 后可正常关闭 server 并清理状态文件
- 避免固定端口冲突，且在 server 不可用时能给出明确错误提示
