# 任务 003：模板候选检测与视觉标注

## 目标

为指定 slide 生成候选字段列表，并输出带编号框的预览图，辅助 Agent 判断页面是否可作为模板。

## 主要工作

- 基于中间模型识别 text/image 候选对象
- 生成候选 JSON，包含类型、位置、文本摘要等信息
- 输出带编号和颜色区分的标注图
- 设计候选字段的 role hint 规则

## 交付物

- `template inspect` 或等价命令
- 标注图输出能力
- 候选 JSON 输出能力

## 验收标准

- 能对指定页输出清晰的标注图
- JSON 中的 candidate 与图上编号一一对应
- Agent 可以据此判断“是否可作为模板”以及“哪些对象是字段”
